# Vue 组件通讯方式

  1. props / $emit

  2. ref / $refs

  3. provide / inject

  4. EventBus

    // eventbus.js
import Vue from 'vue'
export default new Vue()
// 触发事件
import eventBus from 'eventbus.js'
eventBus.$emit(事件名, 值)
// 监听事件
import eventBus from 'eventbus.js'
eventBus.$on((e) => { //... })

  5. $attrs / $listener

    • attrs：包含了父作用域中不作为 prop 被识别 (且获取) 的 attribute 绑定 (class 和 style 除外)。
    • listener：包含了父作用域中的 (不含 .native 修饰器的) v-on 事件监听器。

    通常配合 inheritAttrs (opens new window) 使用。

    • true：继承除 props 之外的所有属性。
    • false：只继承 class 属性。
    // 父组件
<com-a name="Ertsul" age="26" hobby="music"></com-a>

// 子组件 com-a
<com-b v-bind="$attrs"></com-b>
export default {
	props: { name: '', age: '' }
}

// 子组件 com-b
export default {
	props: { hobby: '' }
}
